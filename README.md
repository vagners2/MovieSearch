# MovieSearch

App de exemplo para consulta de Filmes usando a API Rest 
(http://www.omdbapi.com)

Pre-requisites
--------------

- Android SDK v23
- Android Build Tools v23.0.3
- Android Support Repository
- Android Studio 2.1.2

Gradle / Libs
--------------

- Retrofit 2.1.0
- Picasso 2.5.2
- ButterKnife 8.2.1
- Realm 1.1.0
